# 意休Doit

#### 介绍
> 介绍：GTD清单处理工具，已上传至utools。  
> 使用说明：[图文](https://www.yuque.com/nibaobao-youho/pqbrew/doit)

#### 功能介绍
- 清单处理 （基于GTD的清单分类：收集箱 执行|日历|等待|项目|可能 参考资料 回收箱）
  * 另增卡：检查卡(目标或计划) | 完成卡 | 小本本(随记) + 刷新/定期任务
  * 检查页模块：分析模块(记录进程) | 完成模块 | 记忆模块
  * 辅助面板：标签管理 | 问题记录 | 连接互用 | 项目管理
- 记录任务进程--统计分析数据（自评难度，记录用时，通过完成度提高做事的自估能力）
- 数据统筹--近30天的打卡，任务数和任务完成度初判
- 签到--打卡距离目标更进一步的点
- 自定义数据--根据个人情况做恰当数据呈现

#### 安装教程

- 方法1：下载包后直接打开html即用
- 方法2：线上网站即用 https://kathy_candy.gitee.io/doit
- 方法3：utools插件搜索doit即用

#### 使用说明

- [做事] 温习记忆模块内容，做今日规划，捕获事件将其归类并按重要性排序-完成事项
- [复盘查看] 简版页数据-数据整理-另存 （除杂-清楚什么是没必要的）
- [复盘分析] 分析清单-完成自估，完成清单-每天的量与度是否合理，签到-最特别的点，小本本-小想法，参考资源整理-分析资源，搜索记录&问题记录-分析解决问题的思路
- [周复盘] 签到-打卡方向，检查卡-写中期目标，等待清单（主动出击） 项目清单（主动将其拆解） 可能清单、回收箱（是否有变动）
